UpdateTable {Tetris}R Documentation

Update location of tetriminos

Description

Update Teris figure when a tetromino is dropping

Usage

UpdateTable(gp, tetromino)

Arguments

gp

plot object which contian current backgroup

tetromino

the matrix indicates the location of dropping tetromino

Details

This function will update the figure when location of tetriminos is dropping or rotating.
