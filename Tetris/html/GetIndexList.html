GetIndexList {Tetris}R Documentation

Get index of colored square in a tetromino

Description

Get (x,y) for squares in a tetromino which should be colored.

Usage

GetIndexList(tetromino)

Arguments

tetromino

the matrix indicate the location of a tetromino

Details

This function will write a 3x3 matrix as a specific shape tetromino.
