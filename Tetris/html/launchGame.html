launchGame {Tetris}R Documentation

Launch the Game

Description

launch the Game with shiny app

Usage

launchGame()

Details

This function will launch the Game with shiny app. "W" will rotate the tetrominos, "A" will move the tetrominos to left, and "D" will move the tetrominos to right
